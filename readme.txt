
1. Create bookmark for any site in the browser's toolbar.
2. Edit the bookmark and replace the URL link with:
	javascript:q=document.getSelection();open('http://translate.google.com/#auto|en|'+(q));
	
3. save


Usage

1. Navigate to any webpage
2. Select text string to translate to english
3. Click on bookmarklet link
4. Enjoy translation
